Navigation (requires navigation.js)

Navigation of this page is a good example of usage. Navigation trigger is independent of the navigation itself and can be placed anywhere in the document body - even multiple times (triggered navigation is selected via data-navigation-trigger attribute. For instance this link.

attribute what for
*[data-navigation-trigger] specifies ID of the navigation you want to trigger by clicking this element

Navigation comes with navigation trigger (burger-like menu button):

                    <a href="#" class="navigation-trigger white" title="Show navigation" data-navigation-trigger="nav">
    <span></span>
</a>
