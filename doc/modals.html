Modals (requires modals.js)

attribute what for
*[data-modal] URL that will be used to load contents of the modal, every element with this attribute will become a modal trigger
*[data-size] (combined with data-modal) size of the modal - undefined or one of [tall, wide, medium, large, full]
*[data-close] (combined with data-modal) whether or not to show the close button (default true)

Following code must be present inside the body tag in order to use modal (contents of close button are up to you, this documentation is using a fontawesome icon)!

                    <div id="modal-container" class="modal-container">
    <div class="modal">
        <a href="#" class="close">
            <i class="fa fa-times"></i>
        </a>
        <iframe src="" data-link="" name="modal"></iframe>
    </div>
</div>
                
                    <button type="button" data-modal="partials/modalContent.html" class="btn orange">Basic modal</button>
<button type="button" data-modal="partials/modalContent.html" data-size="tall" class="btn mauve">Tall modal</button>
<button type="button" data-modal="partials/modalContent.html" data-size="wide" class="btn turquoise">Wide modal</button>
<button type="button" data-modal="partials/modalContent.html" data-size="medium" class="btn teal">Medium modal</button>
<button type="button" data-modal="partials/modalContent.html" data-size="large" class="btn yellow">Large modal</button>
<button type="button" data-modal="partials/modalContent.html" data-size="full" class="btn crimson">Full modal</button>
<button type="button" data-modal="partials/modalContent.html" data-close="false" class="btn black">No close button</button
