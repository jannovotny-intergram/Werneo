Dropdowns (requires dropdowns.js)

Dropdowns can be used either as menus (use anchors as dropdown options - <a class="dropdown-option">...</a>) or select boxes (then you will need to use the data-input-id attribute).

attribute what for
.dropdown[data-input-id] id of the input to save selected value to if you use dropdown as select-box
.dropdown[data-submit] if set to true closest parent form will be submited on change (assigned input will be updated first of course)
.dropdown-option[data-value] the value to be saved to the input if you use dropdown as seelct-box
.dropdown-option[data-label] the text that will occur in dropdown if this option is selected (of not set text of this option will be used)
