Dynamic Filters (requires dynaFilter.js)

Dynamic filters allow you to filter loaded content by class (each item must have dynafilter-item class). Triggers may be anything you want (buttons, dropdown, spans, whatever). Targeted dynamic filter is selected via data-target attribute of .dynafilter element. Class to be filtered by is selected via data-class attribute of .dynafilter-trigger element.

attribute what for
.dynafilter[data-target] selector (for querySeletor method) of targeted dynamic filter (any element with dynafilter-item children)
.dynafilter-trigger[data-class] class to filter
