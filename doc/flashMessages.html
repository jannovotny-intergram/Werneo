Flash Messages

Types

Contents

You can put practically anything inside the flash message, it accepts HTML content and message type.

Timeout

Default timeout (delay before a message is automatically hidden) of a message is 5 seconds. You can set the timeout in milliseconds manually when creating a message (see examples below). Use 0 for no timeout (message must be dismissed manually).

Flash messages are created using javascript call (you can try copying following code into console):

                    werneo.plugins.flashMessages.createMessage('any HTML content',null);        // default
werneo.plugins.flashMessages.createMessage('any HTML content','error');     // error
werneo.plugins.flashMessages.createMessage('any HTML content','info');      // info
werneo.plugins.flashMessages.createMessage('any HTML content','success');   // success

werneo.plugins.flashMessages.createMessage('any HTML content',null);        // default timeout (5s)
werneo.plugins.flashMessages.createMessage('any HTML content',null,1000);   // 1s
werneo.plugins.flashMessages.createMessage('any HTML content',null,20000);  // 20s
werneo.plugins.flashMessages.createMessage('any HTML content',null,0);      // no timeout (never hides)
