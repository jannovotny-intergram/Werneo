Gallery (requires gallery.js)

Gallery can alter image container size to make rectangular grid of images.

attribute what for
.gallery[data-gutter] space between items of the gallery in pixels
.gallery[data-max-items-per-line] maximal number of items per line
.gallery[data-min-height] minimal item height
.gallery[data-max-height] maximal item height
.gallery[data-loading-text] text to display while loading images

Mind that some settings are logically mutually exclusive (e.g. if you wanted to have max. 3 items per line each 10px high with 1000px gallery width - max-height will be overriden in this case).
